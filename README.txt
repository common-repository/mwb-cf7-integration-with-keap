MWB CF7 Integration with Keap allows you to integrate your Contact Form 7 submissions with the Keap account. Therefore, you can upload data on Keap effortlessly with negligible errors.


== Description ==

**MWB CF7 INTEGRATION WITH KEAP IS A RELIABLE INTEGRATION OF KEAP (FORMERLY K/A INFUSIONSOFT) WITH FORM SUBMISSIONS. IT IS OF GREAT ASSISTANCE WHEN IT COMES TO GENERATING LEADS AND CONTACTS.**

Integrating Contact Form 7 and Keap (Infusionsoft) is an asset for your website. It effortlessly syncs the forms’ data with the platform. The latest update brings to your connection using a global app for smooth integration. This Keap integration plugin also allows you to generate entries in contacts and various salesforce objects.

Using the plugin, you can create as many feeds as you want. Feeds are the rules based on which the contact form submissions are sent to the Keap modules. This data sync is of high value since it helps you run effective campaigns and convert your leads with minimal effort.

Along with the plugin, you will also get other functionalities to ease your process of form submission  The plugin also provides the user with settings adding more utility and making the form submission data synchronization process effortless. These settings include viewing logs, e-mail notifications for errors, customization of feeds for field mapping, and many more features.

==  WITH CF7 INTEGRATION WITH KEAP PLUGIN, YOU CAN: ==

* Connect your forms with Keap modules using the global app.
* Integrate the Keap fields with contact forms entries. 
* Use conditional filters wherever necessary. These filters employ AND/OR logic for your form entries that are needed to be synced to your Keap dashboard. 
* Enable email notifications in the settings to notify the errors that occur during data synchronization.
* Set a primary key that will automatically update data linked to the same.
* View a comprehensive log of the contact form submissions sent over to the Keap formerly k/a Infusionsoft.
* Turn on the toggle switch to delete all data when the plugin is installed.

== WHAT IS INCLUDED IN FREE  CF7 INTEGRATION WITH KEAP  PLUGIN == 

**1)** **Account Authorization**
Contact forms- Keap integration plugin allows you to easily connect the forms with your Infusionsoft account using either the global app or your own API credentials.


**2)** **Refresh Token in One Click**
The plugin dashboard comes with a refresh button to allow you to refresh the access token with just one click. This saves ample time by saving the time spent authorizing the sync between CF7 fields and the Keap module fields each time you want to use your Keap account optimally.


**3)** **Precise Field Mapping**
You can define as many feeds as you want using the plugin. The plugin allows you to sync your contact form 7 entries with the Keap object fields as per the feeds set by you in the Infusionsoft- CF7 integration plugin. This will also help in avoiding probable errors that might occur during data syncing.


**4)** **Unique Identification Using the Primary Key**
For the Contact Form 7 entries, you can also set a primary key mapped with a Keap module field. The primary key updates the data related to the key for every new entry instead of creating a new one.

**5)** **Notifications of Errors On Email**
This integration plugin allows you to enable email notifications by merely a toggle button. You will get notified of an error during data sync on your email. Every time an error occurs during data synchronization, a thorough fault description is sent to you right there in your inbox.

**6)** **Logs for the Data Sync**
View the logs of the contact forms data synced with the Keap. Besides being able to view these logs, you can also download this log history. The logs will ease the process of finding out errors, bugs, or other discrepancies that might occur during the sync process.

**7)** **Filter the Form Entries to be Synchronized**
You can use the conditional filters given in the MWB CF7 Integration with Keap plugin to filter the entries that you need to update on your CRM using the MWB CF Integration with Keap plugin. These filters work by employing the AND/ OR logic.

== Frequently Asked Questions ==

=  How is the MWB CF7 Integration with Keap plugin helpful? =

It helps you send all the data from contact form submissions & entries over Keap as per the fields of Keap. This assists you in creating simplified data over Keap

= How to set up the MWB CF7 Integration with Keap? =

Create an App in your Keap account. You will get the API key and Secret ID from the same app. Copy these keys to enter them in the respective spaces provided. Click on Authorise and you are good to go. If you are still confused, go through the documentation to understand setup through screenshots.

= Can we filter the submissions by using MWB CF7 Integration with Keap? =

Yes, you can filter CF7 submissions as per feeds given in MWB CF7 Integration with Keap. The feeds allow you to set the filters using AND/ OR conditions.

= Can we also see log details within MWB CF7 Integration with Keap? = 
Yes, we have given an option so that you can view the log details. Enable the logs using the toggle button in the Settings tab.

= If the data is not usable, can we delete it? =
Yes you can delete the plugin data which is no more required.


=  If there is any error, will it be notified? =
If there is any error then you will get an email notification. Just enable the email notifications to toggle from the ‘Settings tab.

= Can we also delete the logs after some days? = 
Yes, you can Delete logs after N days. You can set the number of days after which the logs will automatically get deleted in the ‘Settings tab. You can also click on the ‘Clear Log’ button in the ‘Logs’ tab to delete the logs. This will not affect the already synced data with the Keap CRM.

= Is it possible to download the logs data with MWB CF7 Integration with Keap? =

Yes, you can download logs with MWB CF7 Integration with Keap. Just click on the ‘Download’ button in the ‘Logs’ tab.

= How many data filters can we use? =
You can add “AND”/ “OR” conditional statements while creating a feed. It will act as the conditional filter for the form submissions to be synced with your Keap account.

= How to reauthorize if the plugin is disconnected after some hours? =
Plugin will remain connected but the access token expires after 24 hours so you can reauthorize merely by clicking on the refresh button given in the plugin.
